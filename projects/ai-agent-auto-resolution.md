# ServiceNow AI Agent for Incident Auto‑Resolution

## Overview
Designed and deployed a ServiceNow AI Agent to automate L1 incident triage and resolution, reducing manual workload and improving response times.

## Problem
High‑volume L1 teams were overloaded with repetitive incidents (password resets, access issues, VPN failures). Manual triage caused delays and SLA breaches.

## Solution
- Built a ServiceNow AI Agent that:
  - Classifies incoming incidents  
  - Suggests or executes resolutions using Knowledge Base  
  - Routes complex cases to correct assignment groups  
- Integrated with **Flow Designer** for automated workflows  
- Trained Predictive Intelligence using **10,000+ historical incidents**

## Impact
- **40% reduction** in L1 manual workload  
- **60% faster** response time  
- **25% improvement** in SLA compliance  

## Tech Stack
ServiceNow AI Agents, Predictive Intelligence, Flow Designer, ITSM
