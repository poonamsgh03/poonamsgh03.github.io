# ITSM Ticket Categorization ML Model

## Overview
Developed a machine‑learning model to auto‑categorize ITSM incidents and improve routing accuracy.

## Problem
Incorrect categorization caused misrouting, reassignments, and SLA breaches.

## Solution
- Cleaned **10,000+ incident records**  
- Engineered features from descriptions, assignment groups, and resolution notes  
- Trained a multi‑class classification model  
- Designed integration flow for ServiceNow routing suggestions  

## Impact
- **85%+ classification accuracy**  
- **30% fewer misrouted tickets**  
- **50% faster triage**  

## Tech Stack
Python, Pandas, Scikit‑Learn, Excel, ITSM domain knowledge
