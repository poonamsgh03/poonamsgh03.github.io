# Power BI SLA & Operations Dashboard

## Overview
Built a Power BI dashboard to provide real‑time visibility into ITSM performance and SLA trends.

## Problem
Leadership lacked a unified view of SLA performance and operational bottlenecks.

## Solution
- Consolidated ServiceNow + Excel data  
- Designed dashboards for:
  - SLA compliance  
  - MTTR, MTTA  
  - Backlog trends  
  - Category‑wise distribution  
- Automated weekly reporting  

## Impact
- **20% reduction** in SLA breaches  
- Eliminated manual reporting  
- Enabled proactive resource planning  

## Tech Stack
Power BI, DAX, Power Query, ServiceNow data exports
